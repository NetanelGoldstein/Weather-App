If you would like to query a single date, just enter it in the start date field. You can leave all the other date fields blank.

If you would like to query a single range of dates, such as July 3, 2010 through July 5, 2010, fill out the start date field and the end date field.

If you would like to view that date range over several years, fill out the start date and end date field for the first year. Then, add the latest year you would like to query to the End Year Field. If you wanted to see if it rained around July 4th from 2010 to 2020, enter July 3, 2010 in the Start Date Field, and July 5, 2010 in the End Date Field, like before. Then, simply add the year 2020 to the End Year Field.
